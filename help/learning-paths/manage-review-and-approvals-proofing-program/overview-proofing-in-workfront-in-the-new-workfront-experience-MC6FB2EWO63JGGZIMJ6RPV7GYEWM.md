# Vue d’ensemble – Relecture dans Workfront dans la nouvelle expérience Workfront

Les parcours de formation Workfront One suivants :

* Vue d’ensemble – Relecture dans Workfront dans la nouvelle expérience Workfront
* Réviser et approuver des BAT dans la nouvelle expérience Workfront
