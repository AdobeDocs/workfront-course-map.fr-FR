# Champs calculés et expression dans les formulaires personnalisés

Le parcours de formation Workfront One *Champs calculés et expressions dans les formulaires personnalisés* a été migré vers [Créer des champs calculés dans les formulaires personnalisés](https://experienceleague.adobe.com/?recommended=Workfront-L-1-2022.1.calculatedfields){target="_blank"} sur Experience League.

Le nouveau cours Experience League couvre les mêmes objectifs de formation que le parcours de formation précédent et, lorsque cela a été possible, a été optimisé pour offrir une expérience de formation complète.  Explorez la bibliothèque complète de cours, documentation, tutoriels et communautés relatifs à Workfront sur Experience League.
